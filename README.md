# Magic Books CRUD - Biblioteca Mágica

## Visão Geral

Bem-vindo ao Magic Books CRUD, um aplicativo React desenvolvido para capacitar Dona Clara a gerenciar sua coleção de livros mágicos de forma eficaz e encantadora. Com funcionalidades que abrangem desde a adição de novos livros até a visualização detalhada de cada um deles, o aplicativo foi projetado para tornar a experiência de gerenciar sua coleção tão mágica quanto os próprios livros.

**Visite o aplicativo em ação:** [https://magic-books-crud-biblioteca.vercel.app/](https://magic-books-crud-biblioteca.vercel.app/)

## Funcionalidades Principais

### Adicionar Livro

Dona Clara pode expandir sua coleção mágica facilmente, inserindo detalhes como Título, Autor, Ano de Publicação, Gênero e Descrição. Um mecanismo de validação inteligente impede a inserção de anos de publicação futuros, garantindo a precisão dos registros. Com todos os campos obrigatórios preenchidos, um novo livro se junta à sua coleção com um toque mágico.

### Editar Livro

Manter os detalhes atualizados é simples: Dona Clara pode revisar e editar informações de livros já registrados. A edição envolve os mesmos campos que o formulário de adição: Título, Autor, Ano de Publicação, Gênero e Descrição. No entanto, a data de cadastro e o ID permanecem intocáveis, assegurando a integridade dos registros.

### Excluir Livro

Antes de liberar um livro de sua coleção, Dona Clara é solicitada a confirmar sua escolha. Essa etapa evita exclusões acidentais e permite que a coleção permaneça organizada e completa.

### Visualizar Livros

Uma lista exuberante de todos os livros na coleção de Dona Clara é apresentada em uma elegante grade. Informações essenciais, como Título, Autor e Ano de Publicação, estão sempre à vista. Para uma exploração mais profunda, basta escolher a opção "Detalhes", revelando todos os aspectos mágicos do livro selecionado.

### Estilização de Livropolis

A estética do Livropolis é incorporada aos estilos do aplicativo, criando uma atmosfera encantadora e mágica. Cores suaves, fontes elegantes e ícones reminiscentes de livros aprimoram a experiência visual.

### Funcionalidade de Pesquisa

Para facilitar a localização de um livro específico, uma funcionalidade de pesquisa foi habilmente implementada. Com apenas alguns toques, Dona Clara pode encontrar rapidamente o livro que procura, seja pelo título ou autor.

## Tecnologias Utilizadas

- React: O framework central para construção da interface do usuário.
- Vite: Uma ferramenta de build rápida que impulsiona o aplicativo React.
- Styled-components (possivelmente): Uma biblioteca de estilização em JavaScript para criar estilos encapsulados nos componentes.

## Executando o Aplicativo

1. Clone este repositório: `git clone [(https://github.com/cadonaenrike/MagicBooks-crud-biblioteca)]`
2. Navegue até o diretório do projeto: `cd magic-books-crud`
3. Instale as dependências: `npm install` ou `yarn install`
4. Inicie o aplicativo: `npm run dev` ou `yarn run dev`
5. Acesse o aplicativo em seu navegador: [http://localhost:3000](http://localhost:3000)
